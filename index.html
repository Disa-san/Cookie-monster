Cookies

Not all cookies are for eating!

A cookie is a small file that is stored on the users computer. The file contains information about your visit to the site and is used for technical reasons to improve the website for you as a user. By using and navigating the website you consent to the use of cookies. Some cookies are used to create statistics about the users, in order to improve the usability of the website. If you do not want cookies to be stored on your coputer you can decline when asked, but then there might be limitations in how the website is displayed.
